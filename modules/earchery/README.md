# eArchery module

Simple archery arcade proof of concept for FFTA Modules / Compet+.

## Features

- Moving target in a browser-only arena.
- Click-to-shoot interaction.
- Score calculation from the target position at click time.
- 12-shot round.
- Local best score.
- Basic shot history.
- i18n support.

## Intent

This module is deliberately small. It is a PoC used to validate the idea of an archery-oriented arcade / serious-game feature without introducing a rendering engine, assets, authentication, online leaderboard, or persistence model.

Future iterations can add difficulty levels, moving patterns, achievements, leaderboards, mobile touch tuning, and integration with Compet+ Live events.
