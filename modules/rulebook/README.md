# Rule Book module

Lightweight FFTA rule book companion for `ffta-modules` / Compet+.

The module does not embed the heavy official PDF. It stores a compact searchable index, frequent rule cards and a link to the official PDF.

## Features

- Main table of contents by discipline.
- Quick search across rule cards, tags and article references.
- Favorites stored locally.
- Official PDF link.
- Small footprint suitable for Ianseo module deployment.

## Source

Official document: `Règlements sportifs et arbitrage — Version Mai 2026`.
PDF URL: https://www.fftiralarc.org/DOCUMENTS/ARBITRES/R%C3%A8glements_sportifs_et_arbitrage_saison_2026_version_20260502.pdf
