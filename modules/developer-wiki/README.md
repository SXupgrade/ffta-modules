# Developer Wiki

Embedded documentation module for FFTA Modules SDK.

It provides:

- beginner-friendly simple module guide;
- MVVM module guide;
- qualification score read/write examples;
- Lab testing workflow;
- module certification checklist;
- AI-friendly prompts.

The module is read-only by design and is available both in the Lab and in Ianseo when the user has read access to `developerWiki`.
