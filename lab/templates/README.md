# FFTA Modules Lab templates

These templates are intentionally small and copy/paste friendly.

- `simple-readonly`: manifest-only module using `app.data.entries.list`.
- `simple-readwrite`: manifest-only module with read and write actions.
- `export-helper`: custom module using `app.files` and `app.data`.
- `mvvm-advanced`: starter ViewModel for a full MVVM module.

All variable names, comments and default UI strings are in English. Localized UI should go through i18n files.
